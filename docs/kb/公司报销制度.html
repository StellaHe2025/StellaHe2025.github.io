### §RULE_0001 规则 — 1 总则 ## 1 总则 【锚点: 1_总则】 ### §RULE_0003 规则 ### 1.1 制度目的 ### §RULE_0004 规则 为规范公司费用报销行为，加强财务管理控制，合理控制费用支出，降低运营成本，提高资金使用效率，根据《中华人民共和国会计法》、《企业会计准则》及国家相关财税法规，结合本公司实际情况，特制定本制度 ### §RULE_0005 规则 费用报销管理是公司内部控制的重要组成部分，完善的报销制度不仅能有效防止浪费和虚假报销，还能提升员工满意度和工作效率，为公司稳健发展提供财务保障 ### §RULE_0006 规则 ### 1.2 适用范围 ### §RULE_0007 规则 本制度适用于本公司全体正式员工、试用期员工以及外聘顾问和临时工作人员的各类费用报销管理 ### §RULE_0008 规则 本公司分支机构、子公司应参照本制度制定相应的费用报销实施办法，报总公司财务部备案 ### §RULE_0009 规则 本制度所述费用报销包括但不限于差旅费、业务招待费、办公费、会议费、培训费、通讯费、交通费、采购费等日常经营性支出 ### §RULE_0010 规则 ### 1.3 基本原则 ### §RULE_0011 规则 公司费用报销遵循"**实事求是、及时准确、合规节约、分级负责**"的基本原则 ### §RULE_0012 规则 所有报销必须基于实际发生的业务，票据真实合法，内容填写准确完整，报销及时不得拖延，符合财税法规和公司规定，各级审批人应认真履行职责，确保费用支出合理高效 ### §RULE_0013 规则 *表：费用报销基本原则详解* ### §T001_R01_0014 表格规则 — 实事求是 副总以上人员可单独住宿，其他人员同性二人同时出差，按一个房间标准报支 ### §RULE_0132 规则 **交通标准**：旅途中符合乘卧（从晚八时至次日晨七时之间，在车上过夜6小时以上或连续乘车时间超过12小时）而未乘座卧铺，特快按票价50%，其它列车票价60%予以补助 ### §RULE_0133 规则 *表：差旅费报销标准一览表* ### §T005_R01_0134 表格规则 — 总经理及以上 人员级别：总经理及以上；住宿费标准（元/天）：实报实销；交通标准：头等舱/公务舱/一等座；伙食补助（元/天）：150 ### §T005_R02_0135 表格规则 — 部门总监 人员级别：部门总监；住宿费标准（元/天）：500；交通标准：经济舱/二等座；伙食补助（元/天）：120 ### §T005_R03_0136 表格规则 — 部门经理 人员级别：部门经理；住宿费标准（元/天）：400；交通标准：经济舱/二等座；伙食补助（元/天）：100 ### §T005_R04_0137 表格规则 — 普通员工 人员级别：普通员工；住宿费标准（元/天）：300；交通标准：经济舱/二等座；伙食补助（元/天）：80 ### §RULE_0138 规则 ### 5.2 业务招待费报销 ### §RULE_0139 规则 业务招待费应严格**控制标准和频率**，防止铺张浪费 ### §RULE_0140 规则 所有业务招待必须事前通过审批，报销时需提供正规发票及相关证明材料 ### §RULE_0141 规则 **报销要求**：业务招待费需说明客户名称、接待目的、参与人员，部分企业要求事前报备总经理 ### §RULE_0142 规则 招待费发票内容应具体明确，大类发票需附消费明细 ### §RULE_0143 规则 单次招待消费超过1，000元的，需提供餐厅菜单或消费明细 ### §RULE_0144 规则 **限额标准**：业务招待费实行分级限额制度，普通员工单次招待不超过300元，部门经理单次不超过500元，总监级单次不超过800元，总经理及以上根据实际需要确定 ### §RULE_0145 规则 超过标准的需特别说明并经上级特批 ### §RULE_0146 规则 ### 5.3 办公费与采购报销 ### §RULE_0147 规则 办公费用及物资采购报销应遵循**经济高效、公开透明**的原则 ### §RULE_0148 规则 小额采购（如文具）可简化流程，按月汇总报销 ### §RULE_0149 规则 大额采购必须执行比价或招标程序 ### §RULE_0150 规则 **报销凭证**：采购类费用报销必须附采购合同与验收单，且验收人与经办人不得为同一人 ### §RULE_0151 规则 涉及固定资产采购的，还需办理资产入库手续，附资产入库单 ### §RULE_0152 规则 采购发票内容应与合同一致，金额相符 ### §RULE_0153 规则 **信息化采购**：软件采购、系统开发等信息化支出报销时，需附验收报告和使用部门确认意见 ### §RULE_0154 规则 金额超过10万元的信息化项目，还需提供可行性研究报告和项目效益分析 ### §RULE_0006 规则 — 6 监督与责任 ## 6 监督与责任 【锚点: 6_监督与责任】 ### §RULE_0156 规则 ### 6.1 内部控制与审计 ### §RULE_0157 规则 公司建立**多层次的费用报销内部控制体系**，通过制度防患、过程监控和事后审计相结合的方式，确保费用报销的合规性和合理性 ### §RULE_0158 规则 财务部门应定期对费用报销数据进行统计分析，发现异常情况及时预警报告 ### §RULE_0159 规则 **审计监督**：公司设立定期或不定期的审计机制，检查报销的真实性与准确性 ### §RULE_0160 规则 内部审计部门应每半年对费用报销进行一次专项审计，重点关注业务招待费、差旅费等高风险领域 ### §RULE_0161 规则 审计结果向董事会审计委员会报告，并作为部门考核的重要依据 ### §RULE_0162 规则 ### 6.2 违规处理 ### §RULE_0163 规则 对违反费用报销规定的行为，公司实行**"零容忍"政策**，一经发现严肃处理 ### §RULE_0164 规则 虚假报销、重复报销等恶意行为，不仅追回损失，还将视情节轻重给予相应处罚： ### §RULE_0165 规则 - **轻微违规**：第一次出现票据粘贴不规范、信息填写不全等情况，给予口头警告 ### §RULE_0166 规则 - **一般违规**：超期报销、超过标准报销等情况，要求书面检查并扣减绩效分数 ### §RULE_0167 规则 - **严重违规**：提供虚假票据、虚构业务报销，追回报销款项并处以等额罚款，同时给予记过处分 ### §RULE_0168 规则 - **重大违规**：串通舞弊、金额巨大、涉及刑事犯罪的，解除劳动合同并移交司法机关处理 ### §RULE_0169 规则 ### 6.3 责任追究机制 ### §RULE_0170 规则 公司建立**"谁经办、谁负责，谁审核、谁担责"的追责制度**，明确各个环节的责任人及责任范围 ### §RULE_0171 规则 报销人对费用的真实性和合法性负首要责任，审批人对费用的合理性和必要性负审批责任，财务人员对票据的合规性和数据的准确性负审核责任 ### §RULE_0172 规则 对于因审核不严、监督不力导致公司财产损失的，追究相关人员的连带责任 ### §RULE_0173 规则 对坚持原则、堵住漏洞的员工，公司给予表彰和奖励，营造风清气正的财务管理氛围 ### §RULE_0174 规则 本制度自发布之日起执行，原有规定与本制度不一致的，以本制度为准 ### §RULE_0175 规则 本制度由财务部负责解释和修订，财务部可根据实际执行情况提出调整建议，报总经理办公会批准后实施 ### §RULE_0176 规则 **附件**：1.
